# OFET 3D Viewer (OpenGL)

A lightweight OpenGL-based 3D visualization tool for Organic Field-Effect Transistor (OFET) structures.

---

## Features

- STL-based device visualization  
- Layered OFET structure rendering  
- Transparent dielectric layer  
- Interactive camera control  
- Circuit overlay (VG, VSD, GND)  
- Shadow rendering  
- PNG screenshot export  

---

## Requirements

- Linux (Ubuntu / Debian recommended)  
- OpenGL  
- freeglut
