# OFET 3Dビューア（OpenGL）

有機トランジスタ（OFET）の構造を可視化するための軽量3Dビューアです。

---

## 特徴

- STL形式によるデバイス構造表示  
- OFET多層構造の描画  
- 誘電体の半透明表示  
- マウスによる視点操作  
- 配線表示（VG, VSD, GND）  
- 影表示  
- PNG画像保存  

---

## 動作環境

- Linux（Ubuntu / Debian 推奨）  
- OpenGL  
- freeglut  

---

## 実行方法

```bash
chmod +x run.sh
./run.sh
